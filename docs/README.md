# BEIET Documentation Hub

Welcome to the documentation for the BEIET Discord Tutor Bot. This documentation is structured according to the [Diátaxis Framework](https://diataxis.fr/), which divides documentation into four distinct types based on the user's needs:

## 1. Tutorials (Learning-Oriented)
Lessons that take the reader by the hand through a series of steps to complete a project.

## 2. How-To Guides (Problem-Oriented)
Recipes that guide the reader through the steps required to solve a specific problem.

## 3. Reference (Information-Oriented)
Technical descriptions of the machinery and how to operate it.
- [Database Schema Reference](reference-database.md) *(Pending)*
- [Environment Variables](reference-env.md) *(Pending)*

## 4. Explanation (Understanding-Oriented)
Discussions that clarify a particular topic, broadening the reader's understanding.
